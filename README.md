# china-national-highway
中国国道地理位置数据库  
本项目以GeoJSON格式提供GCJ-02标准的中国国道地图地理位置数据。数据位于db文件夹中。

采集进度：
- [X] G101 北京市-沈阳市
- [ ] G318
- [ ] G219
- [ ] G228
- [ ] G331

### 贡献(采集)方式
本项目集成了基于高德地图jsapi的采集网页：  
1、```git clone https://github.com/unclehking/china-national-highway.git```  
2、```cd china-national-highway.git```  
3、```npm install```  
4、```npm run start```
